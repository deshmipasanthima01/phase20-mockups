# Vocabulary mockups

Interactive concepts for discovering, practising, organising, and retaining vocabulary. Add each prototype as a clearly named HTML file and link it from the main catalogue.

## Current mockups

- [DECIDE Family Journey](decide-family-journey.html) — **Ready for feedback** concept showing five increasingly independent encounters with one word family across a 14-day spaced-review journey. Related areas: Practice Sessions and Student Experience. [Vocabulary Engine feature record](https://app.notion.com/p/39c076bec8c981249e40e141147af3fd).
- [Vocabulary Engine — Build Specification](vocabulary-engine-build-spec.html) — **Ready for feedback** proof of concept covering spaced-repetition scheduling, six practice formats, Add Word enhancements, and an interleaved queue builder. [Feature documentation and pipeline record](https://app.notion.com/p/39b4433581618145b767f1f9d468fa7c).
