01 · REVIEW ENGINE

Spaced-repetition scheduler NEW

Replaces "practice whenever" with a due-date system. Each word gets an interval (days) and ease (multiplier) that adjust after every session score. Below is a real six-session run for the word "develop."

Strong (≥85): interval × ease, ease creeps up.
OK (60–84): interval × 1.3, ease unchanged.
Weak (30–59): interval shrinks (× 0.6), doesn't reset.
Fail (<30): interval resets to 1 day.
function scheduleNextReview(word, sessionScore):
    quality = score >= 85 ? "strong" : score >= 60 ? "ok"
             : score >= 30 ? "weak"   : "fail"

    if quality == "strong":
        word.interval = round(word.interval * word.ease)
        word.ease = min(word.ease + 0.1, 2.8)
    elif quality == "ok":
        word.interval = round(word.interval * 1.3)
    elif quality == "weak":
        word.interval = max(1, round(word.interval * 0.6))
        word.ease = max(word.ease - 0.15, 1.3)
    else:
        word.interval = 1
        word.ease = max(word.ease - 0.2, 1.3)

    word.next_review_date = today + word.interval
    return word
Mastery rule change: currently "ready to master" fires at avg score ≥ 85. Recommend requiring both avg ≥ 85 and interval ≥ 14 days, so a word only masters after the engine has already shown durable recall — not off one lucky session.
